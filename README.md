This is a fork of the Magestorm repo @ https://github.com/Magestorm/Magestorm.

You will need the Arenas.dat, Spells.dat, and Grids folder located in the Content Folder in the directory where the SpellServer.exe file is launched from.

There is a spellbinder-server.sql file in the Content Folder. You can use this to import into MySQL to build the nessecary database tables.
